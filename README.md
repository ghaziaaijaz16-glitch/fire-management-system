# Fire Management System

A Java-based Fire Management System designed to manage fire incident records and improve emergency response coordination.

## Features
- Add and manage fire incident records
- Track emergency responses
- Simple user interface
- Organized data handling

## Technologies Used
- Java
- Eclipse IDE
- Object-Oriented Programming (OOP)

## Project Structure
- src/ → Source code files
- images/ → Screenshots and UI images
